Internet Download Manager 6.33.3 中文特别版(IDM)

Takeit
2019-07-03
软件

提升你的下载速度最多达 5 倍，安排下载时程，或续传一半的软件。Internet Download Manager 的续传功能可以恢复因为断线、网络问题、计算机当机甚至无预警的停电导致下传到一半的软件。此程序具有动态档案分割、多重下载点技术，而且它会重复使用现有的联机，而不需再重新联机登入一遍。聪明的 in-speed 技术会动态地将所有设定应用到某种联机类型，以充分利用下载速度。
